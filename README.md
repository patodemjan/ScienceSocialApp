## Data Protection / Datenschutz
All data used in this application is mock data for demonstration purposes.
No real personal data is collected or stored.

Alle in dieser Anwendung verwendeten Daten sind Fiktion. Es werden keine echten personenbezogenen Daten erfasst oder gespeichert.
